## 思路
+ 先理清比较的轮数
+ 在控制每轮的比较的次数
+ 每轮比较的次数等于轮数减去第i轮
+ 源码
```
 var arr = [10,0,20,30,50,40];
		//轮数
		for( var i = 0 ;i<arr.length-1;i++){
			//控住每1轮比较的次数
			for( var j =0;j<arr.length-1-i;j++ ){
				if(arr[j]>arr[j+1]){
					var st =arr[j];
					arr[j]= arr[j+1];
					arr[j+1] =st;
				}
			}
		}
		console.log(arr);
```
